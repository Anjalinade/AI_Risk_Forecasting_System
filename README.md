# AI_Risk_Forecasting_System
Machine learning prototype for loan risk forecasting using Logistic Regression

## Overview
This project predicts the risk level of loan default using a Logistic Regression model.
The system outputs a risk probability and categorizes it as HIGH, MEDIUM, or LOW.
